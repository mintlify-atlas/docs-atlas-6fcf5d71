LoopBack 4 is a highly extensible Node.js and TypeScript framework that lets you build modern REST APIs and microservices quickly. It combines a powerful IoC container, automatic OpenAPI 3.0 specification generation, and a database-agnostic repository pattern — so you spend less time on boilerplate and more time on business logic.

## Why LoopBack 4?

LoopBack 4 was redesigned from the ground up in TypeScript with three goals: extensibility, type safety, and minimal boilerplate.

## Get up and running

<Steps>
  <Step title="Install the CLI">
    ```bash
    npm install -g @loopback/cli
    ```
  </Step>
  <Step title="Scaffold a new app">
    ```bash
    lb4 app
    ```
    Answer the prompts to name your project and select features. The CLI sets up TypeScript compilation, ESLint, and a ping endpoint automatically.
  </Step>
  <Step title="Add a model, datasource, and repository">
    ```bash
    lb4 model
    lb4 datasource
    lb4 repository
    ```
    LoopBack generates the files and wires up the dependency injection for you.
  </Step>
  <Step title="Start the server">
    ```bash
    npm start
    ```
    Visit `http://127.0.0.1:3000` to see the API Explorer with your live OpenAPI spec.
  </Step>
</Steps>

<Note>
  LoopBack 4 requires Node.js 18, 20, or 22. Use the Active LTS release (v22) for production workloads.
</Note>
